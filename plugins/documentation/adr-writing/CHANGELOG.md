# Changelog

## [1.0.0] - 2025-02-07

### Added
- Initial release of adr-writing plugin
- `adr-creating` skill with creation and validation modes
- Support for simple (Context/Decision/Consequences) structure
- Support for multi-domain (domain-by-domain) structure
- YAML front matter generation with area validation
- 5 progressive disclosure reference files
- Self-validation against 8 required coverage items
- Structured validation reports with severity levels
- Shopware-specific pattern support (feature flags, cross-references, audience split)
